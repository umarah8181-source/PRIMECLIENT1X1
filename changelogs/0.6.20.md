## 📋 Launcher Changelog v0.6.20

### ✨ New Features

#### V3 Profile Redesign
- **Redesigned Profile Detail View (V3)** - Completely reworked profile detail UI with hero section and consistent design language
- **Inline-Edit Chips** - Edit group, Minecraft version, loader, and loader-version directly from the profile hero
- **V3 Worlds & Servers Tab** - New dedicated tab for singleplayer worlds and multiplayer servers
- **V3 Screenshots Tab** - Screenshot browser integrated into profile detail
- **V3 Logs Tab** - Log viewer as a profile tab
- **BrowseContentSideSheetV3** - New side sheet for browsing mod/resource-pack content

#### Internationalization (i18n)
- **Full Translation System** - Complete i18n prototype → v2 → v3 rollout [#2582](https://github.com/PrimeClient/issues/issues/2582)
- **New Languages** - Spanish, French, Italian, Polish, Turkish, Portuguese (BR), Russian, Hindi
- **Language Flags** - Flag icons for language selection
- **Auto Translation Sync** - `sync-translations.yml` workflow for automatic translation updates

#### Performance
- **Faster Minecraft Boot** - Cache Minecraft natives for subsequent launches
- **Faster Asset Scanning** - `scan_existing_assets` uses in-memory hashmap instead of filesystem checks
- **Improved Source Speed Check** - NRC assets source speed check optimized
- **Launch-Time Profiling** - New `measure_time` macro to find bottlenecks

#### Profiles
- **Pin Profiles** - Pin favorite profiles to the top [#2303](https://github.com/PrimeClient/issues/issues/2303)
- **Playtime Tracking** - Track per-profile playtime + new disk-size command
- **Rework Profile Logs** - Completely reworked log tab [#2495](https://github.com/PrimeClient/issues/issues/2495)
- **Auto-Resolve Loader Version** - Latest loader version auto-resolved from live APIs
- **Per-Loader Version Override Map** - Separate stored loader version per loader with legacy compatibility
- **Deletion Dialog for Mods** - Confirm dialog before deleting mods

#### Loader Support
- **NeoForge MC 26.x Support** - New version scheme support (drops "1." prefix)
- **Custom Forge/NeoForge Core Mod** - Externally load mods so the `/mods` folder stays untouched, and remove duplicate mods

#### Capes
- **Cape Review Status + Tooltips** - See approval status directly [#2473](https://github.com/PrimeClient/issues/issues/2473)
- **Cape Template Downloads** - Download template with or without elytra variant
- **Mandatory Cape Guidelines Acceptance** [#2534](https://github.com/PrimeClient/issues/issues/2534)
- **Staff Cape Deletion** [#2611](https://github.com/PrimeClient/issues/issues/2611)
- **Remove Cape Resize** [#2610](https://github.com/PrimeClient/issues/issues/2610)

#### Analytics
- **Full Launcher Analytics** - Anonymous usage analytics with proper consent banner
- **Default Opt-Out** - Analytics disabled by default, requires explicit accept
- **Generic trackEvent** - Fire-and-forget tracking with real OS info

#### Other
- **Website Sign-In Deeplink** - Use launcher to sign into website via deeplink [#2635](https://github.com/PrimeClient/issues/issues/2635)
- **Router Error Boundary** - Custom error boundary catches render crashes
- **Discord RPC Refactor** - Shared Discord RPC for launcher AND client
- **Code Signing Certificate** - Windows builds are now code-signed
