## 📋 Launcher Changelog v0.6.16

### ✨ New Features
- **New Log Window** - Completely redesigned log window with improved UI [#2412](https://github.com/PrimeClient/issues/issues/2412)
- **Mod Gallery Preview** - Preview for mod images/screenshots [#2404](https://github.com/PrimeClient/issues/issues/2404)
- **Affiliate/Referral System** - Referral links for friends, servers & partners [#1512](https://github.com/PrimeClient/issues/issues/1512)
- **GitHub OAuth Sync** - Synchronization with GitHub [#2431](https://github.com/PrimeClient/issues/issues/2431)
- **Debug Section** - Better Discord support for helpers
- **Log Throttle** - Improved log performance with instant stop feedback
- **Snow Effect** re-added 🎄
