## 📋 Launcher Changelog v0.6.18

### ✨ New Features
- **Friends & Chat** - Full implementation of friends system and chat functionality
- **Notification Bell** - Added notification system with mark as read functionality [#2030](https://github.com/PrimeClient/issues/issues/2030)
- **NRC Version Indicator** - Star icon shows which Minecraft versions and loaders support Prime client [#2421](https://github.com/PrimeClient/issues/issues/2421)
- **Import Progress Bar** - Display progress when importing large modpacks [#2460](https://github.com/PrimeClient/issues/issues/2460)
- **Child Protection Handling** - Popup and detection for Xbox child protection settings, with option to continue without NRC features [#2481](https://github.com/PrimeClient/issues/issues/2481)
- **Improved Referral Links** - Cleaner referral URLs (nrc.gg/invite)

### 🐛 Bug Fixes
- Fixed crash when account is null (ignore_child_protection_warning) [#2481](https://github.com/PrimeClient/issues/issues/2481)
- RAM bar in logs window now uses correct max value [#2480](https://github.com/PrimeClient/issues/issues/2480)
- Fixed panic on Korean characters in logs [#2476](https://github.com/PrimeClient/issues/issues/2476)
- Cape sorting by newest now works correctly [#2472](https://github.com/PrimeClient/issues/issues/2472)
- Curseforge featured search fixed [#2448](https://github.com/PrimeClient/issues/issues/2448)
- Curseforge mod loader type filter fixed
- Log window now comes to foreground when clicking instance button [#2511](https://github.com/PrimeClient/issues/issues/2511)
- Fixed multiple update downloads when clicking update button repeatedly [#2452](https://github.com/PrimeClient/issues/issues/2452)
- Proper error message when not enough disk space during profile import [#2463](https://github.com/PrimeClient/issues/issues/2463)
- Discord link in launcher fixed

### 🔧 Improvements
- Improved chat message loading and scroll behavior
- UI components are disabled when child protection is enabled
- Better handling of child protection API calls
